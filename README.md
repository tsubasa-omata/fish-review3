## サービス概要

- サービス名  
Fish-Review
- 概要  
ユーザーが魚の食味を投稿できるサービス。魚の味は季節や取れた場所などよって味が大きく異なる。ユーザーの投稿によって場所や季節ごとに魚の味が比較できるようにする。主に魚釣りをする人や料理人などがターゲット。

## サービスURL  
https://fish-review3.herokuapp.com

## 技術スタック  
 - Ruby 2.6.3
 - Rails5.2.4
   - Slim　（HTMLテンプレートエンジン使用）
   - Rspec
 - MySQL  
 - heroku
   - S3 (画像データの保存)
