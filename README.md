## サービス概要

サービス名  
 - Fish-Review
 
概要  
 - ユーザーが魚の食味を投稿できるサービス。魚の味は季節や取れた場所などよって味が大きく異なる。ユーザーの投稿によって場所や季節ごとに魚の味が比較できるようにする。
*現在魚種一覧を一部しか表示しておりません。(データベースへのクエリ数制限の為)

想定ユーザー

 - 釣り人
 - 料理人
 - 魚が好きな人全般

目的

 - 市場の評価を受けていない未利用魚の発見と認知
 - 魚の知見を広げる（産地、時期、ごとに美味しい魚がわかる等）
 - 単純に食味の比較が出来て楽しい
